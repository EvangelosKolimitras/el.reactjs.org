### Πώς μπορώ να γράψω σχόλια στο JSX; {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Τα σχόλια πάνε εδώ */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* Λειτουργεί επίσης
   για σχόλια πολλαπλών γραμμών. */}
  Hello, {name}! 
</div>
```
